Built Weekender as a polished responsive React app with:

- Destination comparison cards with budgets, travel times, ratings, and voting
- Filters for budget/travel time
- Group voting pulse and live winner updates
- Lisbon itinerary preview
- Invite/share modal with copy interaction
- Notification state
- Desktop sidebar and mobile bottom navigation
- Self-contained CSS artwork with no external services

Checks passed:

- `npm run check`
- `npm run build`

Run locally with `npm run dev`.
