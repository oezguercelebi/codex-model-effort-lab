# Weekender benchmark starter

This is the frozen neutral starter for `weekender-v1`.

Available tools:

- React and TypeScript
- Vite
- Tailwind CSS
- Lucide React icons

Commands:

- `npm run dev`
- `npm run typecheck`
- `npm run build`
- `npm run check`

Benchmark runs may edit application files but must not add packages or contact external services.
